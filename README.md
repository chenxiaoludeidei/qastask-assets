*   任务描述：

你是一名的从事量子线路优化的科研人员。你的任务是基于以下算法描述，设计并实现一套抗噪量子架构搜索（NA-QAS）框架。

【核心算法描述】

1\. 可变深度搜索空间

架构由多个 Layer 组成，深度可变。对于一个n比特的系统，每层包含：

\- 旋转门层：每个比特独立从 {Rx, Ry, Rz}中选择一个旋转门；

\- 纠缠门层：包含 0 到多个 CNOT 门（控制位和目标位任意组合）。

通过笛卡尔积将旋转门组合空间与纠缠门组合空间相结合，得到每层电路的完整搜索空间。

2\. 混合参数共享策略 (Hybrid Supernet)

为了避免重复训练每个采样的架构，你需要实现包含K个独立经典线性层（专家网络）的 Supernet。训练时采用 \$\epsilon\$-greedy 策略：每次前向传播计算所有专家的 Loss。以 \$1-\epsilon\$ 的概率选择 Loss 最小的最佳专家，以 \$\epsilon\$ 的概率随机选择一个专家。随后仅对被选中的专家以及共享的量子参数进行梯度更新。

3\. 多目标优化 (NSGA-II)

使用 NSGA-II 算法搜索最优架构。适应度函数有两个最小化目标：

目标1：任务 Loss（即所选专家网络输出的预测误差）。

目标2：硬件开销 \$C(A) \= \alpha \cdot N\_{CNOT} + \beta \cdot N\_{depth}\$，即CNOT门数量和线路深度。

【任务要求与约束】

1\. 必须基于 Python 和 MindQuantum 框架实现。

\- 你需要自己设计项目的目录结构和类/函数抽象。

\- 运行结束后，生成best\_circuit.txt文件，记录包含：搜索出的最佳线路结构的分类准确率/基态能量和误差、线路深度和CNOT数量、最佳线路结构的索引、旋转门的角度。

2\. 噪声约束

基于MindQuantum框架设计一个噪声模拟器，噪声包含单比特门和双比特门的比特翻转噪声、去极化噪声、热驰豫噪声，噪声参数按照主流超导量子计算机的基准设置。在训练阶段和搜索阶段均包含噪声约束。

3\. 实验验证要求

该算法通过量子机器学习（二分类和鸢尾花分类）和分子基态计算（LiH）进行验证，二分类和鸢尾花分类通过分类准确率验证量子线路性能，分子基态计算通过基态误差验证量子线路性能。并对比含噪声情况下和理想无噪声情况下的线路结构和线路性能。
